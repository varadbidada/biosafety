# Agent Summary

## Stack
- **Backend**: Python 3.11+ / FastAPI / uvicorn on port 8000
- **Frontend**: React 19 + TypeScript + Vite + react-chartjs-2 + Leaflet + Vitest
- **Design**: Replit .pen file in `designs/biosafety.pen`
- **Data**: Synthetic data in `data/` (generated by `scripts/seed.py`)
- **ML**: XGBoost + LSTM ensemble models in `ml/`

## Key files

### Backend (`api/`)
- `main.py` — FastAPI app (routes, middleware for CORS/cache/rate-limit, structlog)
- `schemas.py` — Pydantic models
- `config.py` — Settings via env vars

### Frontend (`frontend/src/`)
- `App.tsx` — Root component: landing page (default view) + dashboard. View switch via state.
- `components/LandingPage.tsx` — Hero + feature cards + CTA (enter dashboard)
- `components/ErrorBoundary.tsx` — Error boundary with retry
- `components/EnhancedMap.tsx` — Leaflet map with heatmap, clustering, 4 basemaps
- `components/Tooltip.tsx` — Reusable info popover
- `components/Skeleton.tsx` — Shimmer loading placeholders (SkeletonMap, SkeletonMetric, SkeletonChart)
- `types/api.ts` — TypeScript request/response types
- `tests/` — Vitest tests (App, ErrorBoundary)

### ML (`ml/`)
- `train.py` — XGBoost + LSTM training pipeline
- `predict.py` — Ensemble inference
- `evaluate.py` — Performance metrics

### Config
- `frontend/tsconfig.json` — strict TS with path aliases
- `.github/workflows/` — CI (lint, typecheck, test, API test, build)

## Known issues
- LSTM model file is gitignored/commented out in `predict.py` due to size
- `model_version.py` has unicode handling for Windows

## Build & test
```bash
cd frontend && npm run build    # production build
cd frontend && npx vitest run   # run tests
cd api && python -m uvicorn api.main:app --reload  # dev server
```
